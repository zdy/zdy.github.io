自我介绍

本人即将从山东某某大学毕业，去浙江某某大学攻读研究生，方向是网络安全，大学我的专业是软件工程，大学期间呢，我也没咋闲着，参加了三年的ACM竞赛，这个竞赛真的是难，不过对提升编程能力和思维能力是有很大帮助的，不过对我这个菜鸡来说，在这个竞赛上有所成就，真的是有难度。在任何一个地方，只要自己努力做好每个工作就好，建立这个博客也是为了一切从新出发，做好自己，努力学习技术，克制自己的惰性。

成长历程

大学期间就做了几件事，学习+竞赛+项目，参加过ACM竞赛，ASC超算大赛，CCSP竞赛，CCCC团体天梯赛，还有PAT考试，CSP考试，做过的项目都是些网站项目，有javaweb项目，python的Django项目等等，参加竞赛的过程中，我对编程也是产生了极大的兴趣，coding和debug成了我比较喜欢干的事情，不过大量业余时间还是在看剧啊什么的，哈哈哈。
自己大学期间也不是一事无成，靠自己努力保研了，当时在学校打ACM竞赛的时候，不会的题目一般去csdn查题解，当时自己也去开了个csdn开始写，目的很简单，就是当成自己的笔记一样进行写，每次比赛前翻一翻，博客没有加入自己的思想，只是简单记录一下思路等等，之后写的学习笔记会在两个博客同时更新，博客相关链接存放在主页的友情链接中。

交代目的

我建立这个博客目的非常简单：

  1. 记录学习过程，将来自己也能够回顾，能够快速回忆起这些知识点；
  2. 记录自己的心路历程，将来可以给自己留下一个回忆；
  3. 如果能够帮助到别人，我也很高兴；
  4. 写博客的过程，可以不用考虑太多东西，也可以当作休息的过程，我之前的想法感觉写这个东西浪费太多时间了，但是我觉得学完之后，不做一下记录，那么之后很快就忘掉，见到又得找资源重学；
  5. 该博客的一些内容出现错误啊什么的，欢迎通过发邮件来通知修改；
